About the logo

The logo of this blog is composed of a $\rhd$ buffer logic gate and a $\Sigma$ uppercase sigma to represent respectively the strong relationships between computer science, data science, and applied mathematics : the buffer accumulates the knowledge in the input before propagating it to the sigma to generate a synthesized output.
